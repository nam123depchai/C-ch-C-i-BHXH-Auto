🚀 BHXH Auto-Helper & Buster Solver🌟 Tổng QuanBHXH Auto-Helper là giải pháp cứu cánh cho những ai phải tra cứu hàng nghìn mã BHYT thủ công. Công cụ tích hợp script tự động hóa và tiện ích giải captcha thông minh.Tốc độ: Tự động điền dữ liệu chỉ trong 1s.Thông minh: Hỗ trợ giải Captcha bằng giọng nói qua Buster.Tiện lợi: Quản lý danh sách kết quả trực tiếp trên giao diện web.📦 Yêu Cầu Cài ĐặtBạn cần cài đặt "Bộ đôi hoàn hảo" này để công cụ hoạt động:Tampermonkey: Cài đặt tại đây (Dùng để chạy Script).Buster Captcha Solver: Cài đặt tại đây (Dùng để vượt Captcha).🛠 Hướng Dẫn Thiết Lập (3 Bước Gọn Nhẹ)1️⃣ Cài đặt ScriptMở Tampermonkey -> Dashboard -> Create a new script.Dán nội dung từ file script.js vào và nhấn Ctrl + S.2️⃣ Cấu hình BusterMở tiện ích Buster, vào phần cài đặt.Đảm bảo bạn đã cài đặt "Client app" (nếu Buster yêu cầu) để đạt tỉ lệ giải captcha 100%.3️⃣ Cấp quyền File (Quan trọng)[!TIP]Vào Quản lý tiện ích -> Tìm Tampermonkey -> Bật "Cho phép truy cập vào các URL tệp" (Allow access to file URLs) để script không bị lỗi khi load dữ liệu.🎮 Cách Sử Dụng (Workflow)BướcHành độngKết quả01Copy dữ liệu từ ExcelChuẩn bị danh sách tra cứu02Dán vào bảng HelperScript tự động phân tách Họ tên, Mã số03Click biểu tượng BusterCaptcha tự giải trong 2-3 giây04Nhận kết quảDữ liệu tự lưu vào bảng thống kê bên dưới📸 Ảnh Minh Họa(Bạn nên chụp 1 tấm ảnh giao diện bảng điều khiển của bạn rồi dán link vào đây để người dùng dễ hình dung)🤝 Đóng GópNếu bạn thấy công cụ này hữu ích, đừng quên tặng mình 1 ⭐ Star để ủng hộ nhé!Dự án được phát triển nhằm mục đích hỗ trợ cộng đồng làm BHXH nhanh chóng hơn.
